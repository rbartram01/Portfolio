My Skills

HTML & CSS

I learnt how to write HTML and CSS by purchasing website templates and re-writing the original HTML and CSS code to suit the client. Due to me re-writing or removing areas of code, I would have to remove unnecessary stylesheets that were no longer needed. I learnt how to code in HTML and CSS quite quickly using this method.

JQUERY

As well as re-writing HTML and CSS code in the website templates we purchased I would also replace the original slider and replace it with a bootstrap jquery slider. I started to use this slider in quite of few of the websites I was involved in which meant that I became quite familiar with what javascript files I needed to include and remove any javascript files remaining for the previous slider.

JAVASCRIPT

I recently completed an online course 'Build Websites from Scratch' where I learnt how to write areas of javascript and how javascript worked. In my final project I learnt how to manipulate images using javascript, I have included this feature on my portfolio. I found javascript to be a bit of a challenge and therefore I have continued to learn javascript to strengthen my knowledge.

AJAX & PHP

With the more recent websites I have created, I would use ajax, javascript and php to help with submitting contact forms. I had to re-write areas of javascript to link to the correct fields in the contact form. Ajax was used in the processing of the contact form. PHP was used to validate the contact forms before being sent.

BOOTSTRAP

Throughout my career as a web developer I have always used bootstrap. Due to more and more end users visiting websites on mobile devices, I had to make sure that my websites worked well on all mobile devices.

WORDPRESS

I have experience working with wordpress as we had a client who requested we make changes to their website so that they were able to amend the content and upload any images or documentation when necessary themselves.

GIT

Whilst completing my online course I learnt how to use Git. I found it quite interesting and easy to get to grips with. I continued to use Git throughout the rest of my course.

GITHUB

While completing my course I had to complete static website projects. I have uploaded some of these to my Github account, these can be seen here.
